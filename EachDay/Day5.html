What is JavaScript?

JavaScript is a web programming language that brings interactivity and functionality to websites.While HTML structures the content and CSS styles it,Javascript adds dynamic behaviour.

Why JavaScript was introduced?

Javascript was introduced in 1995 by Brendan Eich at Netscape to bring interactivity and dynamic behaviour to web pages.

  • To allow developers to add interactive features like form validation, image sliders and dropdown menus that could run directly in the browser.
  • To enable real-time updates to web pages without needing to reload the entire page,enhancing user experience.
  • JavaScript was designed to be easy to use, making it accessible for developers and even those without a deep programming background.
  • Unlike server-side languages like PHP,JavaScript runs on the client side,reducing the load on servers and speeding up web interactions.
  • It provided a way to make web pages more engaging,functional,and responsive to user actions.

    let titleH1 = document.getElementById("title");
    titleH1.style.color = "red";
    titleH1.innerText = "Numbers form 1-500";
    
    let numbersDiv = document.getElementById("numbers");
    let numbersHtml = "";
    for (let i = 1; i<= 500; i++) {
        numbersHtml = numbersHtml + "<h2 class='badge text-bg-warning'> " + i + "</h2>";
    }
    numbersDiv.innerHTML = numbersHtml;
