What is HTML?

HTML stands for Hypertext Markup Language.It is the standard markup language for creating web pages.HTML consists of a series of elements and tags which tells the browser how to display the content. Html is developed by Tim Berners-Lee in 1991,which was introduced to share information on the newly created WWW.After then,different versions of Html are introduced by including more advanced features, such as support for multimedia and interactive elements.

Why HTML?

  • HTML was introduced to solve the need for a standardized way to share and structure information on the internet.
  • Before its creation, there was no universal system for organizing content online.
  • HTML enabled the creation of web pages that could be linked together hyperlinks, forming the foundation of the internet.
  • HTML provided a common structure for creating web pages so that they could be understood and displayed consistently by different web browsers.
  • HTML was designed to be simple and straighforward, making it accessible even for begineers to create and share content online.
  • It allowed for the inclusion of text, images , videos, and links, making the internet interactive and multimedia-rich.

Html syntax:

  • <!DOCTYPE HTML>:declares the document type.
  • <html>:root element of HTML document.
  • <head>:includes metadata,styles and scripts.
  • <title>:Defines the title displayed on the browser tab.
  • <body>:displays the visible content like text, images and links.
  • <h1>:represents heading of a page.
  • <p>define paragraph.

HTML Syntax

  • Structural Tags:Define the main structure of the document.
  • Heading Tags:Define headings, from the largest (<h1>) to the smallest (<h6>)
  • Text formatting tags:Style or format text.
  • Media Tags:Display images, videos, and other media. 'img' is self closing tag(<img/>)
  • Link and List Tags:Add links and organize content.
  • Table Tags:Create tables
  • Form Tags:Collect input from users.

HTML Attributes:

HTML attributes provide additional information about HTML elements and help define their properties or behaviour.They are always included in the opening tag of an element.

  1. id:Specifies a unique identifier for an element. Example:<div id="header"></div>
  2. class:Assigns one or more class names to an element(used for CSS styling or JavaScript). Example:<p class="intro text-bold"></p>
  3. style:Add inline CSS styling to an element. Example:<h1 style="color:blue;">Hello!</h1>
  4. title:Provides extra information(often displayed as tooltip when hovered). Example:<a href="#"title="More info">Link</a>
  5. href:Specifies the URL of the link. Example:<a href="https://example.com">Visit</a>
  6. src:Specifies the source file location. Example:<img src="image.jpg"/>
  7. alt:Provides alternative text if the image cannot be displayed. Example:<img src="image.jpg" alt="A cool image"/>
  8. type:Specifies the input type,such as text,password,email etc. Example:<input type="text"/>
  9. placeholder:Provides a short hint inside the input field. Example:<input type="email" placeholder="Enter email"/>
