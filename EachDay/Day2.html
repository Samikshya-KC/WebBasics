What is CSS?

CSS or Cascading Style Sheets,is a powerful tool used for designing and styling websites.It allows you to control the layout,colors,fonts,spacing and more of web pages.

Why we use CSS?

CSS can be used for many purposes related to the look and feel of your page.

  • Text styling,for example,for changing the color and size of headings and links.
  • Creating layouts,for example,turning a single column of text into a multiple-column layout.
  • CSS makes websites adaptable to different screen sizes and devices using techniques like media queries.
  • Enables hover effect,transitions,animations and other interactive features that enhance user experience.

How to use CSS?

Using CSS is straightforward and involves applying styles to your HTML content.

  1. Inline CSS

    2. You can apply styles directly to individual HTML elements using the style attribute in the tag.

  2. Internal CSS

    3. Add a <style>block within the <head>section of your HTML file to define styles for multiple elements.

  3. External CSS

    4. Create a separate .css file to hold all your styles, and link it to your HTML file using the <link> tag in the <head>section.

CSS Syntax

CSS syntax is structured to apply styles to HTML elements in an easy-to-read format.

Selector Example Example description
#id #firstname Selects the element with id="firstname"
.class .intro Selects all elements with class="intro"
* * Selects all elements
element p Selects all elements <p>
element,element,.. div,p Selects all elements <div> elements and all <p> elements.
